* |测试主机的大小端字节序|
* 大端：数据低位存放在 高地址
* 小端：数据低位存放在 低地址

* 数据结构超过1字节才有区别

* 测试方法
* 存一个int型数据 0x12345678
* 判断数据低位存放的地址位置高地
* 例： 
* 大端： 地址低位 第一个字节存放高位数据 (*(char*)&num)=0x12
* 小端： 地址低位 第一个字节存放低位数据 (*(char*)&num)=0x78
